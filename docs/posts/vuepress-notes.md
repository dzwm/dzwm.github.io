# VuePress 站点搭建记录

VuePress 适合把 Markdown 内容组织成长期可维护的静态站点。Theme Hope 在博客场景里补齐了分类、标签、归档、阅读时间和深色模式等能力。

## 为什么先做本地预览

本地先看页面密度、色彩和文章卡片效果，能避免部署后再反复调整视觉细节。

## 初版保留的能力

- 中文站点配置
- 首页视觉图和博客卡片
- 分类、标签、归档
- 深浅色切换
- 复制代码和 Markdown 增强
